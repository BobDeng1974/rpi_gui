RpiGui is a lightweight framework to create GUI without X Server on a RaspberryPI with Python 2.7 bindings

I added some HW libraries such as:
* LCD HD44780 (I2C)
* RDA5807 FM radio receiver (I2C)
* TDA7419 audio processor (I2C)

Dependencies:
* libSDL2
